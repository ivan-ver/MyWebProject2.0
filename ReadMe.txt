Назначение кнопок на домашней странице:

Detail list - список всех деталей с пейджингом 10 штук. Также там расположено поле поиска, функционал обновления и удаления детали.
Type list - таблица, сгруппированная по типам деталей. Имеются ссылки на список деталей определённого типа.
Add detail - добавление новой детали.
